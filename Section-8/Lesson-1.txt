Sayfa üzerinde değişecek ya da değişmesini istediğimiz yapılara state adı veririz. 

Kullanacağımız state'leride ya constructor içerisinde tanımlarız ya da düz state olarak yanımlarız mesela:

constructor(props){
    super(props);
    state = {
        name: "zafer"
    }
} ya da

state = {
    name:"zafer"
} olarak tanımlarız.
